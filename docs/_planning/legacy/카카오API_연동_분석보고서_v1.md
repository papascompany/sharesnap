# 카카오 API 연동 분석 보고서
## ShareSnap 공유방 × 카카오톡 연동 가능성 조사

**작성일:** 2026-04-13
**목적:** 카카오톡 사용자들이 쉽게 사진을 공유방에 업로드할 수 있는 방법 조사 및 제안

---

## 1. 결론 요약 (Executive Summary)

> **카카오톡 오픈채팅방/채널을 API로 자동 개설하거나, 채팅방 내 사진을 외부로 가져오는 것은 공식 API로 불가능합니다.**
>
> 다만, 카카오 로그인 + 카카오톡 공유 API + 카카오톡 채널 챗봇을 **조합**하면 카카오톡 사용자들이 자연스럽게 ShareSnap으로 유입하여 사진을 업로드하는 동선을 설계할 수 있습니다.

---

## 2. 카카오 API 기능별 분석

### 2.1 오픈채팅 API — ❌ 불가

| 항목 | 상태 |
|------|------|
| 오픈채팅방 생성 API | **미제공** |
| 오픈채팅방 메시지 수신 API | **미제공** |
| 오픈채팅방 사진 추출 API | **미제공** |
| 오픈채팅 봇 공식 API | **미제공** (방장봇/오픈채팅봇만 존재, 커스텀 불가) |

카카오 데브톡에서도 오픈채팅 관련 API 제공 계획에 대한 질문이 반복적으로 올라오고 있으나, 카카오 측은 우선순위 문제로 계속 지연 중이라 답변하고 있습니다. LOCO Protocol 등 비공식 방법은 약관 위반으로 영구정지 위험이 있어 사용 불가합니다.

### 2.2 카카오톡 채널 API — ⚠️ 부분 가능 (1:1 채팅만)

| 기능 | 가능 여부 | 설명 |
|------|-----------|------|
| 채널 생성 | ✅ 수동 | 카카오톡 채널 관리자센터에서 수동 생성 (API 자동 생성 불가) |
| 1:1 채팅 시작 | ✅ | `Kakao.Channel.chat()` JS SDK로 채팅방 진입 유도 |
| 채널 추가(친구) | ✅ | `Kakao.Channel.addChannel()` 간편 추가 |
| 챗봇 연동 | ✅ | 카카오 i 오픈빌더로 챗봇 구축 가능 |
| 그룹 채팅방 개설 | ❌ | 채널은 1:1 채팅만 지원 |
| 채팅 내 사진 수신(Webhook) | ❌ | 1:1 채팅의 사진을 Webhook으로 받는 기능 미제공 |
| 이미지 업로드 유도 | ⚠️ | 챗봇에서 웹뷰 링크 제공 → 사용자가 웹에서 업로드 |

**제한사항:** 카카오톡 채널은 "비즈니스 ↔ 사용자" 간 1:1 관계이며, "사용자 ↔ 사용자" 그룹 소통은 지원하지 않습니다. 또한 비즈니스 채널 전환(심사 3~5일)이 필요합니다.

### 2.3 카카오 로그인 API — ✅ 완전 활용 가능

| 기능 | 가능 여부 | 설명 |
|------|-----------|------|
| OAuth 2.0 소셜 로그인 | ✅ | 카카오계정으로 간편 로그인 |
| 프로필 정보 취득 | ✅ | 닉네임, 프로필 사진 (동의 기반) |
| 이메일 취득 | ✅ | 비즈앱 등록 시 가능 |
| 친구 목록 조회 | ⚠️ | 사용 권한 별도 신청 필요 |
| Supabase Auth 연동 | ✅ | Supabase가 카카오 OAuth Provider 공식 지원 |
| NextAuth.js 연동 | ✅ | KakaoProvider 공식 지원 |

### 2.4 카카오톡 공유(공유하기) API — ✅ 핵심 활용 포인트

| 기능 | 가능 여부 | 설명 |
|------|-----------|------|
| 친구/채팅방에 링크 공유 | ✅ | Feed, List, Commerce 등 다양한 템플릿 |
| 커스텀 메시지 템플릿 | ✅ | 이미지+제목+설명+버튼 구성 가능 |
| 웹훅 (공유 성공 알림) | ✅ | 공유 성공 시 서버로 알림 수신 |
| 이미지 업로드 | ✅ | `Kakao.Share.uploadImage()` 가능 |
| REST API 지원 | ❌ | JS SDK만 지원 (프론트엔드 전용) |

### 2.5 카카오톡 메시지 API — ⚠️ 제한적

| 기능 | 가능 여부 | 설명 |
|------|-----------|------|
| 나에게 보내기 | ✅ | 사용자 본인 채팅방에 메시지 |
| 친구에게 보내기 | ⚠️ | 사용 권한 별도 신청 + 동의 + 일 한도 제한 |
| 대상: 앱 가입 친구만 | ⚠️ | 같은 앱에 가입한 카카오 친구만 발송 가능 |

## 3. 제안: 카카오톡 연동 전략

### 전략 A: "카카오 로그인 + 카카오톡 공유" 조합 (⭐ 추천)

카카오톡 내에서 직접 사진을 수집하는 대신, **카카오톡을 유입 채널로 활용**하고 **ShareSnap 자체 웹앱에서 사진을 업로드**하는 구조입니다.

```
┌─────────────────────────────────────────────────────────────┐
│                    사용자 플로우                              │
│                                                             │
│  1. 모임장이 ShareSnap에서 공유방 개설                        │
│     └→ 카카오 로그인으로 가입/로그인                           │
│                                                             │
│  2. 카카오톡 공유 API로 초대링크 전송                          │
│     └→ "제주도 가족여행 사진 공유방" 카카오톡 메시지           │
│     └→ 이미지 썸네일 + "사진 올리기" 버튼                     │
│     └→ 클릭 시 ShareSnap 공유방 웹페이지로 이동               │
│                                                             │
│  3. 초대받은 사람이 링크 클릭                                  │
│     └→ 카카오 로그인 (원클릭)                                 │
│     └→ 자동으로 공유방 입장                                   │
│     └→ 모바일에서 카메라/갤러리로 사진 업로드                  │
│                                                             │
│  4. 새 사진 업로드 시 알림                                    │
│     └→ 카카오톡 메시지(나에게 보내기)로 알림                   │
│     └→ "OO님이 사진 5장을 올렸어요!" + 바로가기 버튼          │
│                                                             │
│  5. 포토북/인화 주문                                         │
│     └→ ShareSnap 내에서 편집 → PDF → 주문                    │
│     └→ 카카오톡으로 주문 완료 알림                            │
└─────────────────────────────────────────────────────────────┘
```

**구현 방법:**

```javascript
// 1. 카카오 로그인 (Supabase Auth)
// Supabase Dashboard > Auth > Providers > Kakao 활성화
const { data, error } = await supabase.auth.signInWithOAuth({
  provider: 'kakao',
  options: {
    redirectTo: `${window.location.origin}/auth/callback`,
    scopes: 'profile_nickname profile_image',
  }
});

// 2. 카카오톡 공유 (초대 링크 전송)
Kakao.Share.sendDefault({
  objectType: 'feed',
  content: {
    title: '제주도 가족여행 사진방',
    description: '우리 여행 사진을 함께 모아봐요!',
    imageUrl: 'https://sharesnap.app/og/room-preview.jpg',
    link: {
      mobileWebUrl: `https://sharesnap.app/join/${shareCode}`,
      webUrl: `https://sharesnap.app/join/${shareCode}`,
    },
  },
  buttons: [
    {
      title: '사진 올리기',
      link: {
        mobileWebUrl: `https://sharesnap.app/join/${shareCode}`,
        webUrl: `https://sharesnap.app/join/${shareCode}`,
      },
    },
  ],
});

// 3. 새 사진 알림 (나에게 보내기)
// REST API로 사용자 본인에게 알림 메시지 발송
const response = await fetch('https://kapi.kakao.com/v2/api/talk/memo/default/send', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'Content-Type': 'application/x-www-form-urlencoded',
  },
  body: new URLSearchParams({
    template_object: JSON.stringify({
      object_type: 'feed',
      content: {
        title: 'OO님이 사진 5장을 올렸어요!',
        image_url: thumbnailUrl,
        link: { web_url: roomUrl, mobile_web_url: roomUrl },
      },
    }),
  }),
});
```

**장점:**
- 카카오톡의 높은 보급률 활용 (가입 장벽 최소화)
- 카카오톡 공유 메시지가 자체적으로 바이럴 효과
- Supabase Auth가 카카오 로그인 공식 지원
- 추가 비용 없음 (무료 쿼터 범위)

**단점:**
- 사진 업로드는 ShareSnap 웹에서 수행 (카카오톡 앱 내에서 직접 불가)
- 카카오톡 ↔ ShareSnap 간 앱 전환 필요

### 전략 B: "카카오톡 채널 챗봇 + 웹뷰" 조합

카카오톡 채널에 챗봇을 연결하고, 챗봇이 웹뷰를 띄워 사진 업로드를 유도하는 방식입니다.

```
┌─────────────────────────────────────────────────────────────┐
│  사용자 플로우                                               │
│                                                             │
│  1. ShareSnap 카카오톡 채널 추가 (친구)                       │
│  2. 채널 1:1 채팅에서 "공유방 만들기" 입력                     │
│  3. 챗봇이 공유방 생성 웹뷰 링크 제공                         │
│  4. 웹뷰에서 방 설정 → 생성 완료                              │
│  5. 챗봇이 초대 링크 카카오톡 공유 버튼 제공                   │
│  6. 초대받은 사람: 링크 클릭 → 웹뷰에서 사진 업로드            │
│  7. 챗봇을 통해 알림/주문 안내                                │
└─────────────────────────────────────────────────────────────┘
```

**구현 요건:**
- 카카오 i 오픈빌더에서 챗봇 시나리오 설계
- 비즈니스 채널 심사 통과 (3~5일)
- Skill Server(Webhook) 구축 → ShareSnap 백엔드와 연동

**장점:**
- 카카오톡 앱 내에서 대부분의 동선이 완료됨
- 채널 추가 시 마케팅 메시지 발송 가능

**단점:**
- 비즈니스 채널 심사 필요
- 챗봇 시나리오가 복잡해지면 UX 저하
- 그룹 채팅 불가 (1:1만)
- 챗봇으로 사진을 직접 수신하는 것은 불가 (웹뷰 우회 필수)

### 전략 C: "PWA + 카카오톡 공유" 하이브리드 (⭐⭐ 최종 추천)

ShareSnap 웹앱을 **PWA(Progressive Web App)**로 만들어 모바일 홈 화면에 설치하게 하고, 카카오톡은 순수하게 **초대/알림 채널**로만 활용합니다.

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  ShareSnap PWA (홈 화면 설치)                                │
│  ┌─────────────────────────────────────────────────────┐    │
│  │  • 카카오 로그인 (원클릭)                             │    │
│  │  • 공유방 개설/참여                                   │    │
│  │  • 채팅 + 사진 업로드 (실시간)                        │    │
│  │  • 갤러리 + 포토북 편집 + PDF + 주문                  │    │
│  │  • Push Notification (웹 푸시)                        │    │
│  └─────────────────────────────────────────────────────┘    │
│         │                              │                     │
│         │ 초대 링크 전송               │ 주문 완료 알림       │
│         ▼                              ▼                     │
│  ┌─────────────────┐          ┌────────────────┐            │
│  │  카카오톡 공유   │          │  카카오톡 알림  │            │
│  │  (Feed 메시지)  │          │  (나에게 보내기)│            │
│  │  "사진 올리기"  │          │  "주문 완료!"  │            │
│  │   버튼 포함     │          │   버튼 포함    │            │
│  └───────┬─────────┘          └────────────────┘            │
│          │ 클릭                                              │
│          ▼                                                   │
│  ShareSnap PWA 자동 실행 (설치된 경우)                        │
│  또는 모바일 웹 브라우저로 열림                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**핵심 구현 포인트:**

```
1. PWA manifest.json 설정
   - 홈 화면 아이콘 설치
   - standalone 모드 (주소창 숨김)
   - 카메라/갤러리 접근 가능

2. 카카오 로그인
   - Supabase Auth → Kakao OAuth Provider
   - 최초 1회 로그인 후 세션 유지

3. 카카오톡 공유 (초대)
   - Kakao.Share.sendDefault() → Feed 타입 메시지
   - 딥링크: sharesnap.app/join/{shareCode}
   - OG Image 자동 생성 (공유방 제목 + 대표 사진)

4. 실시간 채팅/사진 공유
   - Supabase Realtime (자체 구현)
   - 카카오톡 채팅과 별개로 앱 내 채팅

5. Push Notification
   - Web Push API (서비스 워커)
   - 새 사진 업로드 시 참여자에게 푸시

6. 사진 업로드
   - 모바일: <input type="file" capture="environment">
   - 다중 선택 + 드래그&드롭
   - 업로드 중 진행률 표시
```

**이 전략을 추천하는 이유:**
- 카카오톡 API의 제한을 우회하면서도, 카카오톡 사용자 경험을 최대한 활용
- PWA로 네이티브 앱에 가까운 UX 제공 (홈 화면 아이콘, 전체화면, 푸시 알림)
- 카카오톡 공유 메시지의 바이럴 효과 활용
- 자체 채팅으로 사진+코멘트 데이터를 완벽하게 제어 가능 (포토북 연동 용이)
- 앱스토어 심사 없이 즉시 배포 가능

## 5. 카카오 Developers 설정 체크리스트

```
□ 카카오 개발자 계정 등록
□ 애플리케이션 등록 (앱 이름: ShareSnap)
□ 플랫폼 등록
  □ Web: https://sharesnap.app (운영)
  □ Web: http://localhost:3000 (개발)
□ 카카오 로그인 활성화
  □ Redirect URI 등록: https://sharesnap.app/auth/callback/kakao
  □ 동의항목: 닉네임(필수), 프로필 사진(선택), 이메일(선택)
  □ Client Secret 생성 및 활성화
□ 비즈앱 전환 (이메일 동의항목 사용 시)
□ 카카오톡 공유 설정
  □ 메시지 템플릿 생성 (초대용, 알림용)
  □ 웹 도메인 등록
□ (선택) 카카오톡 채널 생성
  □ 비즈니스 채널 전환 (챗봇 사용 시)
  □ 앱에 채널 연결
```

## 6. 개발 계획 수정 사항

기존 개발 계획서에서 **M1(인증 모듈)**과 **M2(공유방 모듈)**에 다음 항목을 추가합니다:

### M1 인증 모듈 추가 사항
```
- Supabase Auth에 카카오 OAuth Provider 설정
- 카카오 로그인 버튼 UI (디자인 가이드 준수)
- 로그인 후 카카오 프로필(닉네임, 사진) 자동 매핑
- 카카오 Access Token 서버 저장 (메시지 API용)
```

### M2 공유방 모듈 추가 사항
```
- 카카오톡 공유 초대 기능
  - Kakao JS SDK 초기화
  - Feed 타입 메시지 템플릿 구성
  - 공유방 OG Image 자동 생성 (방 제목 + 대표사진)
  - 공유 Webhook 수신 → 초대 통계
- 알림 기능
  - 새 사진 업로드 시 → Web Push + 카카오톡(나에게 보내기) 알림
  - 주문 완료 시 → 카카오톡 알림
```

### PWA 설정 (M0 공유 모듈)
```
- next.config.js에 PWA 플러그인 추가
- manifest.json 설정 (아이콘, 테마 색상, standalone)
- Service Worker 등록 (오프라인 지원, 푸시)
- iOS/Android 홈 화면 설치 안내 배너
```
